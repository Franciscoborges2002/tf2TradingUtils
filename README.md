## Overview

**TF2 Trading Utils** is a collection of small browser utilities and scripts to make Team Fortress 2 trading easier and more efficient.

---

## Supported Websites

- [**Scrap.tf Extensions**](./scrap.tf)  
  Custom enhancements for Scrap.tf to speed up banking, raffle participation, and item management.

- [**Backpack.tf Extensions**](./backpack.tf)  
  Useful tweaks and visual improvements for price checking, classifieds, and item listings.

- [**Stntrading.eu Extensions**](./stntrading.eu)  
  Added convenience tools for trading and bulk management.

- [**Steamcommunity.com Extensions**](./steamcommunity.com)  
  Enhancements for user profiles, and inventories.

- [**Steamcommunity.com Tradeoffers Extensions**](./steamTradeOffer/)  
  Enhancements for trade offers.
